# 📻 Brunswick Shores Amateur Radio Club (BSARC) - Web Platform Hub

Welcome to the official source code repository for the **BSARC Website** (`n4gm.org`). This platform is a fast, highly secure, database-free engine built using the **Hugo Static Site Generator** and stylized layout containers.
